## Online Courses

| Course name | Course description | Course URL |
|---|---|---|
| **1. Remote Sensing Basics** |  |  |
| NASA ARSET Fundamentals of Remote Sensing | Self-paced introduction to remote sensing basics through combination of slideshows and recordings | https://appliedsciences.nasa.gov/get-involved/training/english/arset-fundamentals-remote-sensing |
|  Remote Sensing Image Acquisition, Analysis and Applications  |  Covers RS fundamentals, as well as different platforms and sensors. There are also lessons in computational algorithms associated with RS | https://www.coursera.org/learn/remote-sensing  |
| **2. Specialty Topics** |  |  |
| Monitoring the Oceans from Space  |  Introduction to earth observation for ocean-specific observation (highlights Sentinel-3, Jason-3 and future Sentinel-6 missions) | https://www.oceansfromspace.org/  |
|  MIT: Atmospheric Radiation  |  Introduction to the physics of atmospheric radiation and remote sensing including use of computer codes  |  https://ocw.mit.edu/courses/12-815-atmospheric-radiation-fall-2008/  |
| **3. School of Courses** |  |  |
|  EO College (Earth Observation)  |  This website hosts a variety of online free + paid courses that also provide certificates upon completion  |  https://eo-college.org/  |
|  ESRI Training Academy  |  This catalog of courses covers topics from the basics to field operations to 3D visualization--great for GIS + programming work  |  https://www.esri.com/training/catalog/search/  |
| Earth Data Science Courses + Textbooks  |  Hosted by UColorado, this page has a ton of great courses based around remote sensing to earth science  |  https://earthdatascience.org/courses/  |
