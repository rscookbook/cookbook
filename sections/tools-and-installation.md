# Required Tools and Installation Instructions

## Programming Tools
### Python
  - commonly used libraries: rasterio, xarray, rioxarray, earthaccess, geopandas, pyresample, GDAL

### R
  - terra, sf, stars, raster

### Command-line
  - GDAL suite, SNAP

### Cloud-based platforms
  - GEE
  - NASA Earthdata
  - ESA SciHub
